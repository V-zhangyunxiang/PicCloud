# Java IO

## 1.读取文本示例

```java
import java.io.*;

public class EfficientStreamCopy {
    public static void main(String[] args) {
        try (InputStream rawInput = new FileInputStream("input.txt");
             BufferedInputStream bis = new BufferedInputStream(rawInput);
             OutputStream rawOutput = new FileOutputStream("output.txt");
             BufferedOutputStream bos = new BufferedOutputStream(rawOutput)) {
            // 使用8KB缓冲区提高读写效率
            byte[] buffer = new byte[8192];
            int bytesRead;
            
            // 直接流式传输：bis → buffer → bos
            while ((bytesRead = bis.read(buffer)) != -1) {
                bos.write(buffer, 0, bytesRead);
            }
            // 刷新缓冲，确保所有数据写入磁盘
            bos.flush();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```
## 2.IO 下载示例

```java
import java.io.*;
import java.net.*;
import java.util.concurrent.TimeUnit;

public class RobustImageDownloader {
    // 配置参数
    private static final int DEFAULT_TIMEOUT_MS = 10000; // 10秒超时
    private static final int MAX_RETRY_ATTEMPTS = 3;     // 最大重试次数
    private static final int RETRY_DELAY_MS = 2000;      // 重试延迟2秒
    private static final int BUFFER_SIZE = 8192;         // 8KB缓冲区

    public static void main(String[] args) {
        // 示例：下载图片
        String imageUrl = "https://example.com/image.jpg"; // 替换为实际图片URL
        String localFileName = "downloaded_image.jpg";
        
        // 文件保存路径：当前项目根目录下
        String projectRoot = System.getProperty("user.dir");
        String localFilePath = projectRoot + File.separator + localFileName;
        
        System.out.println("图片将保存到: " + localFilePath);
        
        try {
            boolean success = downloadImageWithRetry(imageUrl, localFilePath);
            if (success) {
                System.out.println("✅ 图片下载成功: " + localFilePath);
                
                // 验证文件
                File downloadedFile = new File(localFilePath);
                if (downloadedFile.exists()) {
                    System.out.println("📁 文件大小: " + downloadedFile.length() + " 字节");
                }
            } else {
                System.out.println("❌ 图片下载失败，请检查网络连接或URL地址");
            }
        } catch (Exception e) {
            System.err.println("💥 下载过程发生错误: " + e.getMessage());
            e.printStackTrace();
        }
    }
    
    /**
     * 带重试机制的图片下载方法
     */
    public static boolean downloadImageWithRetry(String imageUrl, String localFilePath) {
        int attempt = 0;
        Exception lastException = null;
        
        while (attempt < MAX_RETRY_ATTEMPTS) {
            attempt++;
            System.out.println("🔄 下载尝试第 " + attempt + " 次...");
            
            try {
                downloadImage(imageUrl, localFilePath);
                return true; // 下载成功
                
            } catch (SocketTimeoutException e) {
                lastException = e;
                System.err.println("⏰ 超时异常: " + e.getMessage());
                
            } catch (ConnectException e) {
                lastException = e;
                System.err.println("🔌 连接异常: " + e.getMessage());
                
            } catch (FileNotFoundException e) {
                System.err.println("❓ 文件未找到或URL无效: " + e.getMessage());
                return false; // 立即返回，重试无意义
                
            } catch (SSLException e) {
                System.err.println("🔒 SSL安全连接异常: " + e.getMessage());
                return false; // SSL问题通常重试无法解决
                
            } catch (IOException e) {
                lastException = e;
                System.err.println("📡 IO异常: " + e.getMessage());
                
            } catch (Exception e) {
                lastException = e;
                System.err.println("⚠️  其他异常: " + e.getClass().getSimpleName() + " - " + e.getMessage());
            }
            
            // 如果不是最后一次尝试，则等待后重试
            if (attempt < MAX_RETRY_ATTEMPTS) {
                System.out.println("⏳ 等待 " + RETRY_DELAY_MS + "ms 后重试...");
                try {
                    TimeUnit.MILLISECONDS.sleep(RETRY_DELAY_MS);
                } catch (InterruptedException ie) {
                    Thread.currentThread().interrupt();
                    System.err.println("⏹️  重试等待被中断");
                    return false;
                }
            }
        }
        
        System.err.println("💥 经过 " + MAX_RETRY_ATTEMPTS + " 次尝试后下载失败");
        if (lastException != null) {
            lastException.printStackTrace();
        }
        return false;
    }
    
    /**
     * 单次图片下载实现
     */
    public static void downloadImage(String imageUrl, String localFilePath) throws IOException {
        // 创建父目录（如果不存在）
        File outputFile = new File(localFilePath);
        File parentDir = outputFile.getParentFile();
        if (parentDir != null && !parentDir.exists()) {
            if (!parentDir.mkdirs()) {
                throw new IOException("无法创建目录: " + parentDir.getAbsolutePath());
            }
        }
        
        HttpURLConnection connection = null;
        
        try {
            // 创建URL连接并配置超时
            URL url = new URL(imageUrl);
            connection = (HttpURLConnection) url.openConnection();
            
            // 设置连接参数
            connection.setConnectTimeout(DEFAULT_TIMEOUT_MS);
            connection.setReadTimeout(DEFAULT_TIMEOUT_MS);
            connection.setRequestProperty("User-Agent", 
                "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36");
            
            // 检查HTTP响应码
            int responseCode = connection.getResponseCode();
            if (responseCode != HttpURLConnection.HTTP_OK) {
                throw new IOException("HTTP请求失败，响应码: " + responseCode + " - " + connection.getResponseMessage());
            }
            
            // 获取文件大小信息（如果可用）
            String contentLength = connection.getHeaderField("Content-Length");
            long fileSize = contentLength != null ? Long.parseLong(contentLength) : -1;
            System.out.println("📦 文件大小: " + (fileSize > 0 ? fileSize + " 字节" : "未知"));
            
            // 使用缓冲流进行高效复制
            try (InputStream rawInput = connection.getInputStream();
                 BufferedInputStream bis = new BufferedInputStream(rawInput);
                 OutputStream rawOutput = new FileOutputStream(outputFile);
                 BufferedOutputStream bos = new BufferedOutputStream(rawOutput)) {
                
                byte[] buffer = new byte[BUFFER_SIZE];
                int bytesRead;
                long totalBytesRead = 0;
                
                while ((bytesRead = bis.read(buffer)) != -1) {
                    bos.write(buffer, 0, bytesRead);
                    totalBytesRead += bytesRead;
                    
                    // 简单的进度显示（对于大文件）
                    if (fileSize > 0 && totalBytesRead % (BUFFER_SIZE * 10) == 0) {
                        int progress = (int) ((totalBytesRead * 100) / fileSize);
                        System.out.println("📥 下载进度: " + progress + "%");
                    }
                }
                
                // 确保所有数据写入磁盘
                bos.flush();
                System.out.println("✅ 下载完成，总计 " + totalBytesRead + " 字节");
            }
            
        } finally {
            // 确保连接被断开
            if (connection != null) {
                connection.disconnect();
            }
        }
    }
    
    /**
     * 工具方法：检查文件是否已存在
     */
    public static boolean isFileAlreadyExists(String filePath) {
        File file = new File(filePath);
        return file.exists() && file.length() > 0;
    }
}
```

## 3. URL.openConnection() 和 URL.openStream() 区别
`URL.openConnection()` 和 `URL.openStream()` 是 Java 中用于处理 URL 连接的两个重要方法，它们之间有明显的区别：

### 主要区别

1.**返回值类型不同**
- **`openConnection()`**：返回 `URLConnection` 对象（或其子类，如 `HttpURLConnection`）
- **`openStream()`**：返回 `InputStream` 对象

2.**控制粒度不同**
- **`openConnection()`**：提供完整的连接控制能力
- **`openStream()`**：简单的只读输入流，隐藏了连接细节

### 详细对比

 **URL.openConnection()**

```java
// 使用 openConnection() - 完整控制
URL url = new URL("https://example.com/image.jpg");
HttpURLConnection connection = (HttpURLConnection) url.openConnection();

// 可以设置各种连接参数
connection.setConnectTimeout(10000);      // 连接超时
connection.setReadTimeout(10000);         // 读取超时
connection.setRequestMethod("GET");       // 请求方法
connection.setRequestProperty("User-Agent", "MyApp/1.0"); // 请求头
connection.setDoOutput(true);             // 允许输出
connection.setInstanceFollowRedirects(true); // 跟随重定向

// 获取响应信息
int responseCode = connection.getResponseCode();
String contentType = connection.getContentType();
long contentLength = connection.getContentLength();

// 然后才能获取输入流
try (InputStream inputStream = connection.getInputStream();
     FileOutputStream outputStream = new FileOutputStream("image.jpg")) {
    // 数据传输...
} finally {
    connection.disconnect(); // 需要手动断开连接
}
```

**URL.openStream()**

```java
// 使用 openStream() - 简化版本
URL url = new URL("https://example.com/image.jpg");

// 直接获取输入流，无法设置连接参数
try (InputStream inputStream = url.openStream();
     FileOutputStream outputStream = new FileOutputStream("image.jpg")) {
    // 数据传输...
}
// 无需手动断开连接，底层自动处理
```

### 功能对比表格

| 特性 | `openConnection()` | `openStream()` |
|------|-------------------|----------------|
| **连接控制** | ✅ 完整控制 | ❌ 有限控制 |
| **超时设置** | ✅ 支持 | ❌ 不支持 |
| **请求头设置** | ✅ 支持 | ❌ 不支持 |
| **响应头读取** | ✅ 支持 | ❌ 不支持 |
| **HTTP方法** | ✅ 可设置(GET/POST等) | ❌ 固定为GET |
| **重定向控制** | ✅ 可配置 | ❌ 自动处理 |
| **输出数据** | ✅ 支持(POST数据) | ❌ 只读 |
| **资源管理** | ❌ 需手动disconnect() | ✅ 自动管理 |
| **代码复杂度** | ❌ 较高 | ✅ 简单 |

### 实际应用场景

适合使用 `openConnection()` 的情况：

```java
public class AdvancedDownloader {
    public static void downloadWithDetails(String urlString, String savePath) throws IOException {
        URL url = new URL(urlString);
        HttpURLConnection conn = (HttpURLConnection) url.openConnection();
        
        // 高级配置
        conn.setConnectTimeout(15000);
        conn.setReadTimeout(30000);
        conn.setRequestProperty("User-Agent", "Mozilla/5.0");
        conn.setRequestProperty("Accept", "image/*");
        
        try {
            // 检查响应
            int responseCode = conn.getResponseCode();
            if (responseCode == HttpURLConnection.HTTP_OK) {
                String contentType = conn.getContentType();
                long fileSize = conn.getContentLengthLong();
                
                System.out.println("Content-Type: " + contentType);
                System.out.println("File Size: " + fileSize + " bytes");
                
                // 下载文件
                try (InputStream in = conn.getInputStream();
                     OutputStream out = new FileOutputStream(savePath)) {
                    byte[] buffer = new byte[8192];
                    int bytesRead;
                    while ((bytesRead = in.read(buffer)) != -1) {
                        out.write(buffer, 0, bytesRead);
                    }
                }
            } else {
                throw new IOException("HTTP error: " + responseCode);
            }
        } finally {
            conn.disconnect();
        }
    }
}
```

适合使用 `openStream()` 的情况：

```java
public class SimpleDownloader {
    public static void quickDownload(String urlString, String savePath) throws IOException {
        URL url = new URL(urlString);
        
        // 简单快速的下载
        try (InputStream in = url.openStream();
             OutputStream out = new FileOutputStream(savePath)) {
            byte[] buffer = new byte[8192];
            int bytesRead;
            while ((bytesRead = in.read(buffer)) != -1) {
                out.write(buffer, 0, bytesRead);
            }
        }
    }
}
```

### 性能考虑

- **`openStream()`**：更轻量，适合简单场景
- **`openConnection()`**：功能完整但稍重，适合需要精细控制的场景

### 推荐使用原则

1. **简单读取** → 使用 `openStream()`
2. **需要配置参数** → 使用 `openConnection()`
3. **需要获取元信息** → 使用 `openConnection()`
4. **需要POST数据** → 使用 `openConnection()`

在图片下载场景中，由于通常需要设置超时、User-Agent 等参数，**推荐使用 `openConnection()`** 以获得更好的控制和健壮性。
