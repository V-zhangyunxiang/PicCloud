```java
public ListNode reverseList(ListNode head) {
       ListNode pre = null;
       ListNode cur = head;
       ListNode temp = null;
       while(cur != null){
             temp = cur.next;  // 保存下一个节点
             cur.next = pre;  // 翻转指针

       }
       return pre; //注意返回值
    }
```

```java
双指针法

每步操作需要同时更新指针 pA 和 pB。

如果指针 pA 不为空，则将指针 pA 移到下一个节点；如果指针 pB 不为空，则将指针 pB 移到下一个节点。

public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
     ListNode pA = headA;
     ListNode pB = headB;
     while (pA != pB) {
       pA = pA != null ? pA.next : headB;
       pB = pB != null ? pB.next : headA;
     }
     return pA;  //注意返回值
}
```

# 环形链表

```java
public boolean hasCycle(ListNode head) {
      //快慢指针，乌龟和兔子同时从起点出发。
      ListNode slow = head;
      ListNode fast = head;
      //fast 每次跳 2 步，所以要保证 fast 本身和 fast 的下一跳都不为 null，缺一不可
      while(fast != null && fast.next != null){
         slow = slow.next;   //乌龟走一步
         fast = fast.next.next;//兔子走两步
         if(fast == slow){ //兔子追上乌龟，说明有环
            return true;
         }
      }
      return false;  //一直到了链表末尾，说明无环
}
```

# 链表的中间节点

当快指针遍历完链表时，慢指针就指向链表中间节点。而由于长度为偶数的链表有两个中间节点，因此需要分两种情况考虑：

链表长度为奇数： 当 fast 走到链表「尾节点」时，slow 正好走到「中间节点」。
链表长度为偶数： 当 fast 走到「null」时（越过「尾节点」后），slow 正好走到「第二个中间节点」。

总结以上规律，应在当 fast 遇到或越过尾节点 时跳出循环，并返回 slow 即可。

```java
public ListNode middleNode(ListNode head) {
       ListNode slow = head;
       ListNode fast = head;

       while(fast != null && fast.next != null){
           fast = fast.next.next;
           slow = slow.next;
       }
       return slow;  
 }
```

# [回文链表](https://leetcode.cn/problems/palindrome-linked-list/)

给你一个单链表的头节点 `head` ，请你判断该链表是否为回文链表。如果是，返回 `true` ；否则，返回 `false` 。

```java
  class Solution {
    public boolean isPalindrome(ListNode head) {
        List<Integer> vals = new ArrayList<Integer>();

        // 将链表的值复制到数组中
        ListNode currentNode = head;
        while (currentNode != null) {
            vals.add(currentNode.val);
            currentNode = currentNode.next;
        }

        // 使用双指针判断是否回文
        int front = 0;
        int back = vals.size() - 1;
        while (front < back) {
            if (!vals.get(front).equals(vals.get(back))) {
                return false;
            }
            front++;
            back--;
        }
        return true;
    }
}
```

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

用一个变量记录一个历史最低价格 `minprice`，在第 `i` 天卖出股票能得到的利润就是 `prices[i] - minprice`。遍历价格数组一遍，记录历史最低点，每一天考虑这么一个问题：如果我是在历史最低点买进的，那么我今天卖出能赚多少钱？当考虑完所有天数之时，我们就得到了最好的答案。

```java
 public int maxProfit(int prices[]) {
        int minPrice = prices[0];     //注意初始值
        int maxValue = 0;
        for (int i = 0; i < prices.length; i++) {
            if (prices[i] < minPrice ) {
                //记录最低价格
                minPrice = prices[i]; 
            } else if (prices[i] - minPrice > maxValue) {
            // prices[i] - minPrice 是当天利润，与历史最大利润比较
               maxValue = prices[i] - minPrice;
            }
        }
        return maxValue ;
    }
```

# 二叉树的最大深度

给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

递归的核心就一句话：

> **把大问题，拆成同类型的小问题**

递归的三要素
1️⃣ 终止条件（什么时候停止，直接返回结果）
2️⃣ 拆分子问题（把问题交给下一层）
3️⃣ 整合结果（用子问题的结果，算出当前层的答案）
不要试图在脑子里模拟所有层的调用过程（会绕晕的）

```java
递归法
public int treeDepth(TreeNode root){
    //最底层员工："我下面没人了，深度是0"
   if(root == null){
      return 0;
   }
   //计算左子树最大深度
  int left = maxDepth(root.left);
   //计算右子树最大深度
   int right = maxDepth(root.right)；
   //取最大值，加上我自己这一层
   return Math.max(left , right) + 1;
}

迭代法与层序遍历原理一致
```

# 有效的括号

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

```java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack = new Stack<>();

        for (char ch : s.toCharArray()) {
            // 左括号直接压栈
            if (ch == '(' || ch == '{' || ch == '[') {
                stack.push(ch);
            } else {
                // 是右括号但栈为空，说明没有对应左括号
                if (stack.isEmpty()) return false;
                //取出栈顶左括号来匹配
                char top = stack.pop();
                // 判断是否匹配
                if (ch == ')' && top != '(') return false;
                if (ch == '}' && top != '{') return false;
                if (ch == ']' && top != '[') return false;
            }
        }

        // 栈为空说明所有左括号都被匹配了
        return stack.isEmpty();
    }
}
```

# 最小栈

设计一个支持 `push` ，`pop` ，`top` 操作，并能在常数时间内检索到最小元素的栈。

实现 `MinStack` 类:

- `MinStack()` 初始化堆栈对象。
- `void push(int val)` 将元素val推入堆栈。
- `void pop()` 删除堆栈顶部的元素。
- `int top()` 获取堆栈顶部的元素。
- `int getMin()` 获取堆栈中的最小元素。

```java
class MinStack {

   private Stack<Integer> stack;
   private Stack<Integer> minStack;
    
    public MinStack() {
     stack = new Stack<>();
     minStack = new Stack<>();
     minStack.push(Integer.MAX_VALUE);
    }
    
    public void push(int val) {
       stack.push(val);
       minStack.push(Math.min(minStack.peek(), val));
    }
    
    public void pop() {
       stack.pop();
       minStack.pop();
    }
    
    public int top() {
       return stack.peek();
    }
    
    public int getMin() {
       return minStack.peek();
    }
}
```

# 二叉树的直径

给你一棵二叉树的根节点，返回该树的 **直径** 。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点 `root` 。

```java
class Solution {
    int maxd = 0; //存储当前最大值
    
    public int diameterOfBinaryTree(TreeNode root) {
        depth(root);
        return maxd;
    }
    
    public int depth(TreeNode node){
        if(node == null){
            return 0;
        }
        int Left = depth(node.left);
        int Right = depth(node.right);
        //将每个节点最大直径(左子树深度+右子树深度)与当前最大值比较并取大者
        //二叉树最大深度基础上加上下面这句即可。
        maxd = Math.max(Left + Right, maxd);
        return Math.max(Left,Right) + 1;//返回节点深度
    }
}
```

```java
        if(root == null) {

public TreeNode invertTree(TreeNode root) {
        if(root == null) {
            return null;
        }
        //将二叉树中的节点逐层放入队列中，再迭代处理队列中的元素
        LinkedList<TreeNode> queue = new LinkedList<TreeNode>();
        queue.add(root);
        while(!queue.isEmpty()) {
            //每次都从队列中拿一个节点，并交换这个节点的左右子树
            TreeNode cur = queue.poll();
            
            TreeNode tmp;
            tmp = cur.left;
            cur.left = cur.right;
            cur.right = tmp;
            //如果当前节点的左子树不为空，则放入队列等待后续处理
            if(tmp.left != null) {
                queue.add(cur.left);
            }
            //如果当前节点的右子树不为空，则放入队列等待后续处理
            if(tmp.right != null) {
                queue.add(cur.right);
            }

        }
        //返回处理完的根节点
        return root;
    }

```

# 移动零

| 条件                       | 遍历范围                    | 适用场景                       |

| `fast < nums.length`     | 0 到 length-1，**包含最后一个** | 只访问 `nums[fast]`           |

```java
public void moveZeroes(int[] nums) {
      if(nums == null || nums.length == 0){
           return;
       }
       //先把不是 0 的都挑出来保存，最后在补上所有 0。
       int slow = 0; 
       for(int fast = 0; fast < nums.length; fast++) {
          if(nums[fast] != 0) {
            nums[slow] = nums[fast];
            slow++;         
           }
       }
       for(;slow < nums.length; slow++){
             nums[slow] = 0;
       }
    }
}
```

```java
public int majorityElement(int[] nums) {
        int cand_num = nums[0];
        int count = 1;
        for (int i = 1; i < nums.length; ++i) {
            if (cand_num == nums[i])
                count += 1;
            else if (--count == 0) {
                cand_num = nums[i];
                count = 1;
            }
        }
```

# 数组-二分查找

给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

提示：

- 你可以假设 nums 中的所有元素是不重复的。
- n 将在 [1, 10000]之间。
- nums 的每个元素都将在 [-9999, 9999]之间。

```java
双指针前后遍历
public int search(int[] nums, int target) {
        if(nums == null || nums.length == 0){
             return -1;
        }
        // 避免当 target 小于nums[0] nums[nums.length - 1]时多次循环运算
        if (target < nums[0] || target > nums[nums.length - 1]) {
            return -1;
        } 
        // 定义target在左闭右闭的区间里，[left, right]
        int left = 0;
        int right = nums.length - 1;
        while (left <= right) {
            int mid = (left + right) / 2;
            if (nums[mid] == target)
                return mid;
            else if (nums[mid] < target)
                left = mid + 1;
            else if (nums[mid] > target)
                right = mid - 1;  
        }
        return -1;
    }
```

# 移除元素

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并**原地**修改输入数组。

```java
// 快慢指针
1.检查 nums[fast] 是否等于要移除的值val。
2.如果 nums[fast] 不等于 val，则将 nums[fast] 的值赋给 nums[slow]，表示这个元素是我们需要保留的。然后将 slow 向右移动一位，即 slow++，表示下一个不同值元素应放置的位置。

public int removeElements(int nums[], int val) {
   int slow = 0;
   for(int fast = 0; fast < nums.length; fast++){
       if(nums[fast] != val){
         nums[slow] = nums[fast];
         slow++;
       }
   }
   return slow;
}
```

- `int pop()` 从队列的开头移除并返回元素

- `boolean empty()` 如果队列为空，返回 `true` ；否则，返回 `false`

队列是先进先出，栈是先进后出。

将一个栈当作输入栈，用于压入 push 传入的数据；另一个栈当作输出栈，用于 pop 和 peek 操作。

每次 pop 或 peek 时，若输出栈为空则将**输入栈的全部数据依次弹出并压入输出栈**，这样输出栈从栈顶往栈底的顺序就是队列从队首往队尾的顺序。

```java
class MyQueue {
    Stack<Integer> inStack;
    Stack<Integer> outStack;

    public MyQueue() {
        inStack = new Stack<Integer>();  // 输入栈
        outStack = new Stack<Integer>(); // 输出栈
    }

    public void push(int x) {
        inStack.push(x);
    }

    public int pop() {
        if (outStack.isEmpty()) {
            in2out();
        }
        return outStack.pop();
    }

    public int peek() {
        if (outStack.isEmpty()) {
            in2out();
        }
        return outStack.peek();
    }

    public boolean empty() {
        return inStack.isEmpty() && outStack.isEmpty();
    }

    //关键逻辑。
    private void in2out() {
        while (!inStack.isEmpty()) {
            outStack.push(inStack.pop());
        }
    }
}
```

# LruCache 缓存(中等)

请你设计并实现一个满足 [LRU (最近最少使用) 缓存](https://baike.baidu.com/item/LRU) 约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1` 。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value` ；如果不存在，则向缓存中插入该组 `key-value` 。如果插入操作导致关键字数量超过 `capacity` ，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 必须以 `O(1)` 的平均时间复杂度运行。

```java
LinkedHashMap 维护了一个双向链表，记录插入顺序。
  - 新插入的元素放在链表尾部
  - 重新 put 同一个 key 不改变位置，所以 get/put 时需要先 remove 已有元素再添加
  - 最早插入且没被操作的元素在链表头部

class LRUCache {
    private final int cap;//最大容量
    private LinkedHashMap<Integer, Integer> cache = new LinkedHashMap<>();
    
    public LRUCache(int capacity) {
       this.cap = capacity;
    }
    
    public int get(int key) {
//对于已存在的 key 只更新值不更新位置，所以需要手动删除再重新插入来达到"移到尾部"的效果。
    Integer value = cache.remove(key);
      if(value != null){
          cache.put(key, value);
          return value;
      }
        return -1;
    }
    
    public void put(int key, int value) {
        //删除 key，并利用返回值判断 key 是否在 cache 中
        if(cache.remove(key)!= null){   
           cache.put(key,value);
           return;
        }
        //超出最大容量
        if(cache.size() >= this.cap){
            // 移除头部最久未使用的 key
          int first = cache.keySet().iterator().next();
          cache.remove(first);
        }
        // 将新的 key 添加链表尾部
        cache.put(key, value);
    }
}
```

# 最长连续序列(中等)

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        // 把数组放入 HashSet 去重
        // 后续查找某个数存不存在，时间复杂度是 O(1)
        Set<Integer> num_set = new HashSet<Integer>();
        for (int num : nums) {
            num_set.add(num);
        }

        int longestLength = 0;
        for (int num : num_set) {
        //遍历 num_set 中每个元素，只有当 num-1 不存在时，才把 num 当作起点
            if (!num_set.contains(num - 1)) {
                int currentNum = num;
                int currentLength = 1;
                // 从起点开始往后数
                while (num_set.contains(currentNum + 1)) {
                    currentNum += 1;
                    currentLength += 1;
                }
                // 更新全局最长长度
                longestLength = Math.max(longestLength, currentLength);
            }
        }
        return longestLength;
    }
```

含义为 **前 i+1 个元素的和 = 前 i 个元素的和 + 第 i 个元素，每一层都是在上一层的基础上，加上当前的 nums[i]**。

**作用为在 O(1) 时间内求任意区间 [left, right] 的和**

```java
prefix[0] = 0;
for (int i = 0; i < nums.length; i++) {
    prefix[i + 1] = prefix[i] + nums[i];
}
```

解法 1：

某段子数组的和 = 右边界前缀和 - 左边界前缀和
即：区间和 = total - prefix[i] = k
所以：prefix[i] = total - k

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        // key：不同前缀和的值，value：这个前缀和出现了几次
        Map<Integer, Integer> preSumMap = new HashMap<>();
        // 对于下标为 0 的元素，前缀和为 0，个数为 1
        preSumMap.put(0, 1);

        int preSum = 0;
        int count = 0;
        for (int num : nums) {
          preSum += num;
          // total - k 在哈希表中出现过，意味着存在某个位置 i，使得从 i 到当前位置的子数组和恰好等于 k
          if (preSumMap.containsKey(preSum - k)) {
              count += preSumMap.get(preSum - k);
           }
           preSumMap.put(preSum, preSumMap.getOrDefault(preSum, 0) + 1);
        }
        return count;
    }
}
```

```java
public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        LinkedList<TreeNode> queue = new LinkedList<>();
        if(root != null){
            queue.offer(root);
         } 
         while(!queue.isEmpty()){
            int length = queue.size();
            List<Integer> level = new ArrayList<>();
            while (length > 0) {
                TreeNode node = queue.poll();
```

# 删除链表倒数第 N 个节点(中等)

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

```java
如果我们能够得到的是倒数第 n 个节点的前驱节点而不是倒数第 n 个节点的话，删除操作会更加方便。因此我们可以考虑在初始时将 
slow  指向哑节点，其余的操作步骤不变。这样一来，当 fast 遍历到链表的末尾时，slow  的下一个节点就是我们需要删除的节点。
  
public ListNode removeNthFromEnd(ListNode head, int n) {
       ListNode dummy = new ListNode(0);
       dummy.next = head;
       
       ListNode slow = dummy;
       ListNode fast = head;
       //让快指针先走 k 步
        while(n > 0){
           fast = fast.next;
           n--;
        }
       //快慢指针开始同速前进，快指针走到链表末尾为 Null 时，慢指针所在的位置就是倒数第 k 个链表节点的前驱节点
        while(fast != null){
         slow = slow.next;
         fast = fast.next;
        } 
       //删除倒数第 k 个节点
       slow.next = slow.next.next;
       return dummy.next;  
    }
```

# 环形链表II(中等)

给定一个链表的头节点  head，返回链表开始入环的第一个节点。 如果链表无环，则返回 null

```java
public LinstNode detectCycle(ListNode head) {
      ListNode slow = head;
      ListNode fast = head;
      //获取首次相遇时候，slow 的位置
      while(fast != null && fast.next != null){
         fast = fast.next.next;
         slow = slow.next;
         if(fast == slow)
           break;
      }
      //如果快指针走到尽头，没环
      if (fast == null || fast.next == null) {
         return null;
      }
     // 慢指针指向头结点，快慢指针以相同速度前进，
     // 再次相遇时所在的节点位置就是环开始的位置
     slow = head;
     while(slow != fast){
          slow = slow.next;
          fast = fast.next;
      }
    return slow;
}
```
