```java
public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
     ListNode A = headA;
     ListNode B = headB;
     while (A != B) {
       A = A != null ? A.next : headB;
       B = B != null ? B.next : headA;
     }
     return A;  //注意返回值
}
```

```java
public boolean hasCycle(ListNode head) {
      //快慢指针，乌龟和兔子同时从起点出发。
      ListNode slow = head;
      ListNode fast = head;
      //fast 每次跳 2步，所以要保证 fast 本身和 fast 的下一跳都不为 null，缺一不可
      while(fast != null && fast.next != null){
         slow = slow.next;   //乌龟走一步
         fast = fast.next.next;//兔子走两步
         if(fast == slow){ //兔子追上乌龟，说明有环
            return true;
         }
      }
      return false;  //一直到了链表末尾，说明无环
}
```

# 买卖股票的最佳时机

给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

用一个变量记录一个历史最低价格 `minprice`，在第 `i` 天卖出股票能得到的利润就是 `prices[i] - minprice`。遍历价格数组一遍，记录历史最低点，每一天考虑这么一个问题：如果我是在历史最低点买进的，那么我今天卖出能赚多少钱？当考虑完所有天数之时，我们就得到了最好的答案。

```java
 public int maxProfit(int prices[]) {
        int minPrice = prices[0];     //注意初始值
        int maxValue = 0;
        for (int i = 0; i < prices.length; i++) {
            if (prices[i] < minPrice ) {
                //记录最低价格
                minPrice = prices[i]; 
            } else if (prices[i] - minPrice > maxValue) {
            // prices[i] - minPrice 是当天利润，与历史最大利润比较
               maxValue = prices[i] - minPrice;
            }
        }
        return maxValue ;
    }
```

# 二叉树的最大深度

给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

递归的核心就一句话：

> **把大问题，拆成同类型的小问题**

递归的三要素
1️⃣ 终止条件（什么时候停止，直接返回结果）
2️⃣ 拆分子问题（把问题交给下一层）
3️⃣ 整合结果（用子问题的结果，算出当前层的答案）
不要试图在脑子里模拟所有层的调用过程（会绕晕的）

```java
递归法
public int treeDepth(TreeNode root){
    //最底层员工："我下面没人了，深度是0"
   if(root == null){
      return 0;
   }
   //计算左子树最大深度
  int left = maxDepth(root.left);
   //计算右子树最大深度
   int right = maxDepth(root.right)；
   //取最大值，加上我自己这一层
   return Math.max(left , right) + 1;
}

迭代法与层序遍历原理一致
```

# 有效的括号

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

```java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack = new Stack<>();

        for (char ch : s.toCharArray()) {
            // 左括号直接压栈
            if (ch == '(' || ch == '{' || ch == '[') {
                stack.push(ch);
            } else {
                // 是右括号但栈为空，说明没有对应左括号
                if (stack.isEmpty()) return false;
                //取出栈顶左括号来匹配
                char top = stack.pop();
                // 判断是否匹配
                if (ch == ')' && top != '(') return false;
                if (ch == '}' && top != '{') return false;
                if (ch == ']' && top != '[') return false;
            }
        }

        // 栈为空说明所有左括号都被匹配了
        return stack.isEmpty();
    }
}
```

# 最小栈

设计一个支持 `push` ，`pop` ，`top` 操作，并能在常数时间内检索到最小元素的栈。

实现 `MinStack` 类:

- `MinStack()` 初始化堆栈对象。
- `void push(int val)` 将元素val推入堆栈。
- `void pop()` 删除堆栈顶部的元素。
- `int top()` 获取堆栈顶部的元素。
- `int getMin()` 获取堆栈中的最小元素。

```java
class MinStack {

   private Stack<Integer> stack;
   private Stack<Integer> minStack;
    
    public MinStack() {
     stack = new Stack<>();
     minStack = new Stack<>();
     minStack.push(Integer.MAX_VALUE);
    }
    
    public void push(int val) {
       stack.push(val);
       minStack.push(Math.min(minStack.peek(), val));
    }
    
    public void pop() {
       stack.pop();
       minStack.pop();
    }
    
    public int top() {
       return stack.peek();
    }
    
    public int getMin() {
       return minStack.peek();
    }
}
```

# 二叉树的直径

给你一棵二叉树的根节点，返回该树的 **直径** 。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点 `root` 。

```java
class Solution {
    int maxd = 0;
    public int diameterOfBinaryTree(TreeNode root) {
        depth(root);
        return maxd;
    }
    
    public int depth(TreeNode node){
        if(node == null){
            return 0;
        }
        int Left = depth(node.left);
        int Right = depth(node.right);
        //将每个节点最大直径(左子树深度+右子树深度)与当前最大值比较并取大者
        //二叉树最大深度基础上加上下面这句即可。
        maxd = Math.max(Left + Right, maxd);
        return Math.max(Left,Right) + 1;//返回节点深度
    }
}
```

```java
public TreeNode invertTree(TreeNode root) {
        if(root == null) {
            return null;
        }
        //将二叉树中的节点逐层放入队列中，再迭代处理队列中的元素
        LinkedList<TreeNode> queue = new LinkedList<TreeNode>();
        queue.add(root);
        while(!queue.isEmpty()) {
            //每次都从队列中拿一个节点，并交换这个节点的左右子树
            TreeNode cur = queue.poll();
            
            TreeNode tmp;
            tmp = cur.left;
            cur.left = cur.right;
            cur.right = tmp;
            //如果当前节点的左子树不为空，则放入队列等待后续处理
            if(tmp.left != null) {
                queue.add(cur.left);
            }
            //如果当前节点的右子树不为空，则放入队列等待后续处理
            if(tmp.right != null) {
                queue.add(cur.right);
            }

        }
        //返回处理完的根节点
        return root;
    }

```

# 移除元素

给你一个数组 nums 和一个值 val，你需要**原地**移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并**原地**修改输入数组。

```java
1.检查 nums[fastIndex] 是否等于要移除的值val。
2.如果 nums[fastIndex] 不等于 val，则将 nums[fastIndex] 的值赋给 nums[slowIndex]，表示这个元素是我们需要保留的。然后，将 slowIndex 向右移动一位，即 slowIndex ++，表示下一个不同值元素应放置的位置。
3.如果 nums[fastIndex] 等于 val，则不做交换，直接检查下一个元素，相当于跳过了这个需要移除的元素。
4.遍历结束后，slowIndex 的位置就是新数组的有效长度。  
  
public int removeElements(int nums[], int val) {
   int slowIndex = 0;
   for(int fastIndex = 0; fastIndex < nums.length; fastIndex++){
       if(nums[fastIndex] != val){
         nums[slowIndex] = nums[fastIndex];
         slowIndex++;
       }
   }
   return slowIndex;
}
```

```java
        if (s.length() == 0) 
          return 0;
        HashMap<Character, Integer> map = new HashMap<Character, Integer>();
        int max = 0;
        int start = 0;
        for(int end = 0; end < s.length(); end++){
            char element = s.charAt(end);
            if(map.containsKey(element)){
              //新窗口从重复字符之后的第一个字符开始

              start = Math.max(map.get(element) + 1, start);
            }  
            max = Math.max(max, end - start + 1);
            map.put(element, end);
        }
```

# LruCache 缓存(中等)

请你设计并实现一个满足 [LRU (最近最少使用) 缓存](https://baike.baidu.com/item/LRU) 约束的数据结构。

```java
class LRUCache {
    int cap;
    LinkedHashMap<Integer, Integer> cache = new LinkedHashMap<>();

    public LRUCache(int capacity) {
       this.cap = capacity;
    }
    
    public int get(int key) {
      if(!cache.containsKey(key)){
          return -1;
      }
      makeRecentLy(key);
      return cache.get(key);
    }
    
    public void put(int key, int value) {
        if(cache.containsKey(key)){
            // 更新 key
           cache.put(key,value);
           // 当前 key 更新为最新使用
           makeRecentLy(key);
           return;
        }
        if(cache.size() >= this.cap){
            // 移除头部最久未使用的 key
          int first = cache.keySet().iterator().next();
          cache.remove(first);
        }
        // 将新的 key 添加链表尾部
        cache.put(key, value);
    }

    // 更新位置
    public void makeRecentLy(int key){
        int value = cache.get(key);
        // 删除 key，并重新插入到队尾
        cache.remove(key);
        cache.put(key, value);
    }
}
```

```java
public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        LinkedList<TreeNode> queue = new LinkedList<>();
        if(root != null){
            queue.offer(root);
         } 
         while(!queue.isEmpty()){
            int length = queue.size();
            List<Integer> tmp = new ArrayList<>();
            while (length > 0) {
                TreeNode node = queue.poll();
                tmp.add(node.val);
                if(node.left != null){
                   queue.offer(node.left);
                }
                if(node.right != null){
                   queue.offer(node.right);
                }
                length--;
            }
            res.add(tmp);
         }
         return res;
    }
```

# 环形链表II(中等)

给定一个链表的头节点  head，返回链表开始入环的第一个节点。 如果链表无环，则返回 null

```java
public LinstNode detectCycle(ListNode head) {
      ListNode slow = head;
      ListNode fast = head;
      //获取首次相遇时候，slow 的位置
      while(fast != null && fast.next != null){
         fast = fast.next.next;
         slow = slow.next;
         if(fast == slow)
           break;
      }
      //如果快指针走到尽头，没环
      if (fast == null || fast.next == null) {
         return null;
      }
     // 慢指针指向头结点，快慢指针以相同速度前进，
     // 再次相遇时所在的节点位置就是环开始的位置
     slow = head;
     while(slow != fast){
          slow = slow.next;
          fast = fast.next;
      }
    return slow;
}
```
